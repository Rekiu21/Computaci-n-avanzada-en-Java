# Listas

##subtitulo - practica versiones

En esta práctica creamos una lista de nombres usando `ArrayList<String>` en Java, agregamos varios elementos, los recorrimos para imprimirlos y finalmente mostramos cuántos elementos contiene la lista.

Paso a paso para subir el repositorio a GitHub:

1. Abre una terminal en la carpeta del proyecto.
2. Inicializa Git: `git init`
3. (Opcional) Renombra la rama principal: `git branch -M main`
4. Agrega los archivos a commit (por ejemplo, la carpeta `Listas` y el README): `git add Listas README.md`
5. Crea el commit: `git commit -m "segundo commit"`
6. Agrega el remoto: `git remote add origin https://github.com/Rekiu21/Computaci-n-avanzada-en-Java.git`
7. Sube los cambios: `git push -u origin main`
